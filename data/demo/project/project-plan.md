# Atlas Project Plan

This is fictional demo data created for the MiniRAG Assistant portfolio project.

## Objective

The Atlas project will deliver a secure self-service support portal that helps
customers review requests and helps support teams resolve them efficiently.

## Schedule and milestones

V1 is planned to be completed on 30 September 2026.

The main milestones are requirements approval on 15 July 2026, authentication
integration on 15 August 2026, user acceptance testing on 10 September 2026,
and the V1 completion milestone on 30 September 2026.

## Ownership

The project owner is Maya Chen. The Platform Experience team is responsible for
delivery, with support from the Identity and Security teams.
