# MiniRAG Shared Demo Knowledge Base

This is fictional demo data created for the MiniRAG Assistant portfolio project.

The shared demo knowledge base contains four logical collections:

- `general` provides an overview of the demo.
- `project` describes the fictional Atlas project plan.
- `technical` describes the fictional authentication implementation.
- `policies` contains the fictional security access policy.

Visitors can use these documents to try single-collection routing, Multi-RAG
retrieval, grounded answers, citations, and bounded agent planning.
